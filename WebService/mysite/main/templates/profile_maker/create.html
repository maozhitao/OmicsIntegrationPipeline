How to use it

  1. Fill the topic you are interested in (example: "Salmonella", "stroke")
    NOTE: For the test version, the pipeline will use the experiments in optional experiment filter file and overwrite all query results.
  2. Fill the species you are interested in (example: "Salmonella enterica", "Homo sapiens")
  3. Select a file for correlation validation: a comma-separated values (csv) file with the following columns:
    • "exp_id": Sample ID, the experiment id number in SRA database (example: SRX501844)
    • "series_id": Study ID, the study id number in SRA database (example: SRP040622)
    • "cond_id": Condition label marked by yourself manually (example: "healthy cortex")
    • Example: Salmonella example, Human stroke patients example
  4. Select a sample list file for knowledge capture validation : a comma-separated values (csv) file with the following columns:
    • "exp_id": Sample ID, the experiment id number in SRA database (example: SRX501844)
    • "indicator": 0 or 1, to represent two conditions (example: 0 means healthy people and 1 means stroke patients)
    • Example: Salmonella example, Human stroke patients example
  5. Select a gene list file for knowledge capture validation
    • "gene_list": A list of gene (NOTE: the gene name should be matched with NCBI gene database) (example: MAFK, STM0001)
    • "description" (Optional): The optional description of genes
    • "expected" (Optional): Expected regulation of cases compared with control (example: Up-regulated or Down-regulated)
    • Example: Salmonella example, Human stroke patients example
  6. Select a additional sample filter files (Necessary in test version): a comma-separated values (csv) file with the following columns:
    • "Experiment": A list of gene (NOTE: the gene name should be matched with NCBI gene database) (example: MAFK, STM0001)
    • All other columns for annotation are optional
    • Example: Salmonella example, Human stroke patients example
    NOTE: For the test version, the pipeline will USE ALL the experiments in optional experiment filter file and overwrite all query results.
  7. Provide your email address. The results will send to you.
  8. Provide the project label. (Example: "SalmonellaExample", "HumanExample")
